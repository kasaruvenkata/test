How to call this module (example to put in top-level Infra/main.tf or wherever you run tf apply):

module "monitor_lambda_uat" {
  source        = "./modules/lambda_monitor"
  environment   = "uat"
  function_name = "monitorlambda-uat"
  ecr_repo_name = "monitorlambda-uat-ecr"
  image_tag     = "monitorlambda"
  s3_bucket     = "aabackstop-uat"
  secret_name   = "azneprod"
  aws_region    = "eu-west-1"
  create_role   = true
}


Duplicate for prod with environment = "prod", function_name = "monitorlambda-prod", s3_bucket = "aabackstop-prod", etc.
